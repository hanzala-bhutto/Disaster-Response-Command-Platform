# Learning Map

##  What this project teaches

### Frontend
- React component structure
- dashboard layout
- forms and tables
- API integration

### Backend
- FastAPI routing
- schema validation
- service layers
- clean folder patterns

### Databases
- PostgreSQL tables and relations
- Redis caching basics
- Qdrant vector storage

### Distributed systems
- microservices
- message queues
- event-driven design
- decoupled services

### AI
- embeddings
- RAG pipeline
- prompt structure
- agentic workflows
- orchestration logic

### DevOps
- Docker
- Kubernetes
- configuration and secrets
- Prometheus metrics
- Grafana dashboards
- operational monitoring

## Best way to learn from this project
For every phase, answer these questions:
1. what problem does this phase solve?
2. why is this service needed?
3. what data enters and leaves it?
4. how would the system behave if it fails?
5. what did I simplify and why?
